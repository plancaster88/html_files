Training/Tips

This section shows many of the most important features available in Power BI to effectively navigate and utilize reports.

Export Options
Export data to Excel from a table or visual.

Hover over any visual or table and a toolbar will appear at the top right. Click the ... and choose the Export data option. Click the Export button.


Create PowerPoint of all report pages.

Click File which is on the far left in the toolbar above the report page. Click Export to PowerPoint. Ensure that the Exlude hidden report tabs checkbox is checked. Click the Export button.


Create PDF of all report pages.

Click File which is on the far left in the toolbar above the report page. Click Export to PDF. Ensure that the Exlude hidden report tabs checkbox is checked. Click the Export button.



Interact With Reports
Basic Interaction with Visuals.

Selecting items in visuals or tables will dynamically filter other visuals and tables in the current report page. Simply click on an item or item label to filter other report visuals.

Note: To select multiple items hold Ctrl + click


Using slicers (filters) and date slicers.

Slicers (filters) are a way to filter the report. The PCP dashboard is setup to filter all report pages based on the current slicer selections. There is also a search bar the can be utilized once a slicer is clicked.

Note: To select multiple items in a slicer hold Ctrl + click


Expand visual to fullscreen Focus mode

Hover over any visual or table and a toolbar will appear at the top right. Click the Focus mode button.


Show visual data in table form Show data

Hover over any visual or table and Right Mouse Click. Choose the Show data option.



Drill Into Data
Show detail data Drillthrough

Detail report pages are included for certain visuals and tables. These are known as Drillthrough pages. Right Mouse Click on any item in a report visual or table. Hover over the Drillthrough option. Select the desired detail report.


View next level of data in sub-category format Expand

To view next level of data in a visual as a sub-category of the current category click the + next to an item. You can also Mouse Right Click on the visual, hover over the Expand option and choose either Selection, Entire Level, or All. To drill back up to the initial level of detail click the - next to an item or hover over the visual, Mouse Right Click, and select Drill Up.
Alternatively you can hover over the visual and click the Expand all down one level in the hierarchy button and then the Drill up button to return to the original level of data.

View next level of data without it being in sub-category format Show Next Level

To view next level of data in a visual or table without it being a sub-category Mouse Right Click on the visual and select the Show Next Level option. To drill back up to the initial level of detail hover over the visual, Mouse Right Click, and select Drill Up.
Alternatively you can hover over the visual and click the Go to the next level in the hierarchy button and then the Drill up button to return to the original level of data.

View next level of specific selection by using Drill down

To view next level of a specific selection in a visual or table Mouse Right Click on an item and select the Drill down option. To drill back up to the initial level of detail hover over the visual, Mouse Right Click, and select Drill Up.
Alternatively you can hover over the visual and toggle the Click to turn on Drill Down button and then the Drill up button to return to the original level of data.
