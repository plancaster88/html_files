Getting Started

This section will walk you through how to get started with Power BI reports.

  1. Ensure that you are signed up with Office 365 (O365) by going to https://www.office.com/.
    1. Click Sign In
    2. Type in your Aetna email and then press Next (you should be redirected to O365).

  2. Visit https://app.powerbi.com/ to see if you have access to the Power BI web service.
    1. If not signed in you will have to sign in using step 1. If signed in go through the required steps to activate the Power BI web service.

  3. Once the prior steps are completed you should be able to access Power BI apps which you have been granted permissions. ABHLA Analytics is the primary app for ABHLA Power BI reports. Take the following link: Link to ABHLA Analytics. You should be redirected to the ABHLA Analytics app and land on the PCP Dashboard report. Different reports and report pages can be accessed on the the left-hand navigation menu.

Note: If you need access to the ABHLA Analytics app contact the Power BI web service administrator.
